==========================
TYPO3 Exception 1476109646
==========================

.. include:: /If-you-encounter-this-exception.rst.txt


TYPO3 10.4.36 - 07.04.2023
==========================

Installation Overview
---------------------

Trying to extend `fe_manager` I accidently left the following example code in my :file:`ext_localconf.php`:

.. code-block:: php

    $extbaseObjectContainer = \TYPO3\CMS\Core\Utility\GeneralUtility::makeInstance(\TYPO3\CMS\Extbase\Object\Container\Container::class);
    $extbaseObjectContainer->registerImplementation(
        \In2code\Femanager\Controller\NewController::class,
        \In2code\Femanagerextended\Controller\NewController::class
    );
    
Without implemenign the class or chaninging it to my namespace.

The Issue
---------

I got the - counterintuitive error message: 

..  code-block:: text

    #1476109646 TYPO3\CMS\Extbase\Mvc\Exception\InvalidControllerException
    Invalid controller "In2code\Femanager\Controller\NewController". The controller 
    must implement the TYPO3\CMS\Extbase\Mvc\Controller\ControllerInterface.
    
Even though NewController actually does implement ControllerInterface

Solution
--------

Remove the faulty implementaton registration.
